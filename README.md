# Credit-Card-Fraud-Detection
A Machine Learning model using Random Forest to detect fraudulent credit card transactions. Achieved 97% precision and 78% recall on a dataset of 284k+ transactions.
Dataset: [Link to Kaggle Credit Card Fraud Data]

Key Metrics:

Precision: 0.97

Recall: 0.78

MCC: 0.87

Dataset: Download the CSV file used in this project through https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
